# Example Images for Carousel

This directory should contain example images to showcase in the carousel on the main chat page.

## Required Images

Please add the following images (or similar):

1. **example1.jpg** - A diagram related to physics (e.g., force diagrams, motion graphs)
2. **example2.jpg** - A mathematical graph or visualization (e.g., functions, geometry)
3. **example3.jpg** - A scientific illustration (e.g., biology diagrams, chemistry models)

## Image Requirements

- Format: JPG or PNG
- Recommended size: 500x400px (will be displayed as 250x200px)
- File size: Keep under 500KB each for fast loading

## Temporary Placeholders

SVG placeholders have been provided (example1.svg, example2.svg, example3.svg) for testing.
Replace these with actual educational images for production.
